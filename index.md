## Jekyll-Tailwind-Alpine Starter

This repository provides a Jekyll starter with the following configured:

- Jekyll-postcss plugin
- package.json
    - postcss
    - autoprefixer
    - tailwindcss
    - tailwindcss/typography
    - cssnano
- [Alpine.js](https://alpinejs.dev/) script added to head
